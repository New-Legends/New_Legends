# New_Legends_Project
宁工NewLegends战队工程2022赛季电控_张超

此代码适用于宁波工程学院2022赛季工程机器人顶部MCU

操作方式

左拨杆      右拨杆      效果

上          上          夹爪

上          中          出爪

上          下          翻爪

中          上          矿仓

中          中          抬升

F + 滚轮 = 救援
G        = 地矿
B + 滚轮 = 出卡
Z        = 自动取矿
X        = 自动收矿
C        = 自动出矿
V        = 自动兑换
